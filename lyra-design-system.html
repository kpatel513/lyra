üìã Usage Guidelines

Color Usage

Color Semantics

  • Use success-green for positive metrics
  • Use critical-red sparingly for urgent issues
  • Use lyra-primary for interactive elements
  • Use neutral-slate for secondary text

Avoid

  • Don't use red for non-critical issues
  • Don't mix warm and cool colors randomly
  • Don't use low contrast combinations
  • Don't overuse bright accent colors

Animation Guidelines

Performance First: All animations should enhance usability without impacting performance. Use CSS transforms and opacity for optimal rendering.
